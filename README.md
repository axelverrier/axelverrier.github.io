# My resume
Librement inspiré de [J Glovier](https://github.com/jglovier/resume-template).

**MODIFICATIONS**  
J'ai changé la structure du code en remplaçant les fichiers YAML par des fichiers markdown grâce à la gestion des objets collections de Jekyll.  
Cela permet d'une part une simplification du code, d'autre part une prise en charge des accents et autres caractères spéciaux impossible en YAML.

En outre, j'ai ajouté un bouton qui permet d'afficher une 2e version du contenu de la page (ici une version traduite) grâce à JQuery.
